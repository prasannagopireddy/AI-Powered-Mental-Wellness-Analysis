# Dataset

This project uses the **Reddit Mental Health Dataset** from Kaggle — a collection of Reddit posts
labelled across five mental health conditions: Stress, Depression, Bipolar Disorder, Personality Disorder, and Anxiety.

---

## Download Instructions

1. Go to the Kaggle dataset page:
   👉 https://www.kaggle.com/datasets/neelghoshal/reddit-mental-health-data

2. Click **Download** (you need a free Kaggle account)

3. Extract the ZIP and place the CSV file in your project folder **exactly** like this:

```
AI-Powered-Mental-Wellness-Analysis/
└── Reddit Mental Health Data/
    └── Reddit Mental Health Dataset.csv
```

> ⚠️ The folder name and file name must match exactly, including spaces and capital letters.

---

## After Downloading

Run `preprocess.py` — it will automatically generate the `processed_data/` folder:

```bash
python preprocess.py
```

This creates:

```
processed_data/
├── processed_dataset.csv     ← full cleaned dataset
├── train.csv                 ← 80% training split
├── test.csv                  ← 20% test split
└── label_map.json            ← integer → class name mapping
```

---

## Dataset Details

| Property | Value |
|---|---|
| Source | Reddit (via Kaggle) |
| Total samples | 5,607 posts |
| Number of classes | 5 |
| Split | 80% train / 20% test |
| Input columns used | `title` + `text` (combined) |

### Class Distribution

| Class | Label | Posts |
|---|---|---|
| Stress | 0 | 1,099 |
| Depression | 1 | 1,202 |
| Bipolar Disorder | 2 | 1,085 |
| Personality Disorder | 3 | 1,077 |
| Anxiety | 4 | 1,144 |

---

## Why Is the Dataset Not Included in This Repository?

The raw CSV file is too large for GitHub's file size limits.
The `processed_data/` folder is excluded via `.gitignore` because it is automatically
generated by `preprocess.py` and does not need to be version-controlled.
